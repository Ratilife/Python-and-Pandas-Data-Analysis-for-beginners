Дополнительные функции навигации в PyCharm
Navigate to Class/File/Symbol (Перейти к классу/файлу/символу):

Используйте сочетание клавиш Ctrl + N (для классов), Ctrl + Shift + N (для файлов) или Ctrl + Alt + Shift + N (для символов),
чтобы быстро найти и перейти к нужному классу, файлу или символу.
Recent Files (Недавние файлы):

Нажмите Ctrl + E, чтобы открыть список недавно открытых файлов. Это позволяет быстро переключаться между файлами,
с которыми Вы работали.
Navigate Back/Forward (Перейти назад/вперед):

Используйте Ctrl + Alt + ← и Ctrl + Alt + →, чтобы вернуться к предыдущему месту в коде или перейти вперед.
Structure View (Структура):

Откройте панель структуры с помощью Alt + 7, чтобы увидеть структуру текущего файла. Это позволяет быстро находить методы и переменные.
Search Everywhere (Поиск повсюду):

Нажмите Shift дважды, чтобы открыть окно поиска, где Вы можете искать классы, файлы, символы и даже команды.
Bookmarks (Закладки):

Вы можете добавлять закладки в код, используя F11, и затем быстро переходить к ним с помощью Shift + F11.
Quick Documentation (Быстрая документация):

Нажмите Ctrl + Q, чтобы получить быструю справку о методе или классе, на который Вы наводите курсор.
Refactor (Рефакторинг):

PyCharm предлагает мощные инструменты для рефакторинга кода, такие как переименование, изменение сигнатуры метода и т.д.
Используйте Ctrl + R для доступа к меню рефакторинга.
